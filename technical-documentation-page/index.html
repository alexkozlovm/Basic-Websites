Intro

We’ll cover all the available methods for specifying colors with CSS in this guide. There are six ways to declare CSS colors:

  • Color Names
  • HEX
  • RGB
  • HSL
  • RGBA
  • HSLA
Color Names

CSS has 16 basic color names. Examples of basic color names are blue, maroon, black and white.

div { background-color: blue; }

The code above would result in a div having a blue background color. There are many other CSS color names/keywords you can find a complete list of here.

HEX

HEX, aka Hexadecimal color notation, is a way of declaring a color by indicating the amount of red, green, and blue (RGB) it has. This color notation is based on the hexadecimal numeral system (base 16). We’re more accustomed to the decimal numeral system (base 10), so hex is a bit tricky to understand at first.

div { background-color: #000000; }

In the example above, black was declared as #000000, which means it has no red, green, or blue. (Color theory tells us that the absence of any color is black.)

RGB

The decimal numeral system is what we’re most used to. It’s usually the system we’re taught in school, and it’s how most numbers are displayed in our world. The hexadecimal numeral system, on the other hand, isn’t mainstream; it isn’t used much outside computer programming and math-related fields.

div { background-color: rgb(0, 0, 0); }

The code above would result in a black box.

HSL

HSL stands for Hue, Saturation, and Lightness. The HSL color notation is an alternative to the RGB color notation. It was introduced to CSS with the idea that it’s more intuitive to learn compared to the RGB color system.

div { background-color: hsl(240, 100%, 50%); }

The code above would result in the div having a blue background color.

HSLA

HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity of the color.

div {background-color:hsla(120,100%,50%,0.3);}

The code above would result in the div having a green background color with some opacity.

RGBA

RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity of the color.

div {background-color:rgba(255,0,0,0.3);}

The code above would result in the div having a red background color with some opacity.
